# Respaldos Completos - ProTrabajo

## ¿Qué tienes aquí?

Tienes 2 respaldos completos de tu sitio web ProTrabajo:

### 1. Sitio Web (Astro)
- **Archivo:** `protrabajo-cl-main.zip`
- **Qué es:** Todo el código fuente de tu sitio web
- **Para qué sirve:** Para modificar, actualizar o mover el sitio

### 2. Administración (WordPress)
- **Carpeta:** `wordpress-admin/`
- **Archivos:** 
  - `protrabajo-....zip` (sitio completo)
  - `protrabajo_admin_wpvivid....zip` (administración)
- **Qué es:** El panel de administración donde gestionas contenido
- **Para qué sirve:** Para restaurar el panel de administración

## ¿Cómo usar estos respaldos?

### Si necesitas modificar el sitio web:
1. Descomprime `protrabajo-cl-main.zip`
2. Edita los archivos que necesites
3. Sube los cambios al servidor

### Si el panel de administración falla:
1. Entra a tu hosting
2. Usa el plugin WPVivid
3. Restaura desde los archivos de `wordpress-admin/`
